快速入手

本节对GBM的功能及使用方法进行了简要介绍，帮助用户快速掌握本程序的基本用法。 有关GBM原理与用法的详细介绍请参见详细教程

程序安装

$ tar -zxf GBM.tar.gz

# 安装GBM的Python包依赖：setuptools, wheel, numpy, scipy, pandas, scikit-learn, lightgbm, matplotlib, seaborn
$ pip install setuptools wheel numpy scipy pandas scikit-learn lightgbm matplotlib seaborn

# 安装GBM的外部程序依赖：Plink 1.90 
$ wget s3.amazonaws.com/plink1-assets/plink_linux_x86_64_20191028.zip
$ mkdir plink_1.90
$ unzip plink_linux_x86_64_20191028.zip -d ./plink_1.90


# 将GBM、plink添加到系统环境变量中以便于快速使用：
$ vi ~/.bashrc
export PATH="/userpath/GBM:$PATH"
export PATH="/userpath/plink1.90:$PATH"
$ source ~/.bashrc


参数配置

GBM同时支持读取配置文件与命令行指定参数两种参数赋值形式。

# GBM读取配置文件中各参数的值并调用预处理基因型数据模块
gbm -c config_path -pg

GBM会先读取配置文件中各参数的值，再读取命令行中各参数的值。 运行GBM必须同时指明配置文件的路径。 所以当命令行指定参数与配置文件中参数重复时，GBM会以命令行的值为参考，忽略配置文件中的值。

# GBM会认定输入的基因型数据格式为ped，忽略配置文件中genofile_format的值
gbm -c config_path -pg all --fileformat ped

由于GBM可选参数较多，建议用户根据使用目为配置文件中各参数设置基础值， 利用命令行不断调整少量特定参数的值，以便于达到最佳的分析结果。

基因型数据预处理

基因型数据预处理模块的功能包括根据样本编号、snp编号提取指定基因型数据，数据重编码等。 为程序下游分析提供数据及可接受的文件格式。 目前GBM支持的基因型文件输入格式有ped、bed、vcf和bcv。 其中用户自定义格式需要用户指明多项参数的值，否则程序可能无法从数据文件中正确提取基因型数据。

# 调用预处理基因型数据模块，统计基因型数据的基本情况(杂合率等)
gbm -c config_path -o savepath -pg all --fileprefix genofile --fileformat ped


表型数据预处理

表型数据预处理模块可以实现数据归一化，重编码等功能。 同时支持将处理后数据的分布情况以密度分布图或箱线图的形式进行可视化。

# 调用预处理表型数据模块，对表型数据进行归一化操作
gbm -c config_path -o savepath -pp --phe_norm

# 表型数据可视化
gbm -c config_path -o savepath -pp --phe-plot --phefile-path phefile


群体结构分析

群体结构分析模块可以基于基因型数据分析样本的种群结构。 GBM使用tsne或pca方法对基因型数据进行降维， 使用optics方法对降维后的数据进行聚类，认为聚集到相同类别的样本属于同一群体。 同时支持以散点图的形式可视化样本集的群体结构。

# 调用群体结构分析模块，根据基因型数据对样本进行聚类并可视化
gbm -c config_path -o savepath -s --structure-plot --genofile-path genofile --n-clusters 8


特征选择与表型预测

特征选择与表型预测模块的核心是利用lightGBM模型训练输入数据集， 预测未知表型样本的表型，并筛选出对表型起决定性作用的snp位点。 同时支持以箱形图和热图的形式可视化lightGBM模型所筛选出snp的重要性。

# 对输入的数据进行交叉验证，检验lightgbm模型各参数是否合适
gbm -c config_path -o savepath -e -cv --traingeno genofile --trainphe phefile

# 调用特征选择与表型预测模块，根据输入的训练集数据训练模型并筛选与表型相关性高的snp
gbm -c config_path -o savepath -e -t -sf --bygain-boxplot --traingeno genofile --trainphe phefile

# 用已训练完成的模型对测试集样本进行表型预测
gbm -c config_path -o savepath -e -p --testgeno genofile --modelfile-path modelfile
